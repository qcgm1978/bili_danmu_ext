哔哩哔哩弹幕分析扩展

一个Chrome浏览器扩展，可以在用户访问哔哩哔哩视频页面时获取该视频的弹幕，进行分类，并通过单词云和情绪图表进行可视化展示。

弹幕获取

自动获取哔哩哔哩视频页面上的弹幕数据，无需手动复制粘贴。

情绪分析

对弹幕内容进行情绪分析，区分正面、负面和中性弹幕。

单词云

生成弹幕内容的单词云，直观展示弹幕中高频出现的词汇。

数据统计

提供弹幕数量、情绪分布等统计数据，帮助了解观众反馈。

安装方法

  1. 克隆或下载本项目到本地
  2. 打开Chrome浏览器，输入 chrome://extensions/ 进入扩展管理页面
  3. 开启右上角的「开发者模式」
  4. 点击「加载已解压的扩展程序」，选择项目文件夹
  5. 扩展安装完成，会显示在浏览器工具栏中

使用方法

  1. 打开任意哔哩哔哩视频页面
  2. 点击浏览器工具栏中的扩展图标
  3. 在弹出的菜单中点击「开始分析」按钮
  4. 页面右侧会显示分析面板，包含单词云、情绪分析和统计数据三个标签页
